# Calculator

## Description
A basic web calculator for performing arithmetic operations like addition, subtraction, multiplication, and division.

## Features
- Addition, subtraction, multiplication, division
- Clear button to reset inputs
- Simple and responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation
# Clone the repository
git clone https://github.com/yourusername/calculator.git

# Open the project
cd calculator
# Open index.html in your browser
